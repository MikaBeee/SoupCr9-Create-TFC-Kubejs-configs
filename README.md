# Soupcraft TFC KubeJS Config

This repository integrates **TerraFirmaCraft (TFC)** recipes into **Create**, making Create items craftable in TFC.  

It provides a bridge between the two mods, adjusting recipes to fit TFC progression while keeping the Create content accessible.
